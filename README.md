# weibo_keyword_crawl
Python爬取微博关键词，并且记录相关数据

# 环境安装
1. 需要Python3.8 +的版本

2. 安装依赖环境
```pip install -r requirements.txt```

# 如何使用

1. 配置cookie
在```config.py```文件 ```g_weibo_headers```修改cookie，
cookie从```s.weibo.com```中获取

2.```main.py```中修改```search_config```配置相关搜索信息

3. 运行程序
执行脚本```python main.py```
